# OwnPen — 亲笔写的 — 产品需求文档

> "Make it sound like YOU wrote it."
> "让领导看不出是AI写的"

## 基本信息

- **产品名**: OwnPen — 亲笔写的
- **Slug**: `ownpen`
- **英文**: OwnPen — Make it sound like YOU wrote it
- **定位**: 中文去AI化 + 平台风格化写作工具
- **目标用户**: 所有用AI写中文内容的人（职场人、自媒体、学生）
- **核心痛点**: 领导/读者/平台一眼看出"这是AI写的"

---

## 竞品分析

| 竞品 | 问题 |
|------|------|
| humanizer（英文原版） | 1,366 installs，纯英文规则 |
| humanizer-zh（中文翻译版） | 221 installs，规则全是英文直译，还在检测"-ing结尾"，对中文无效 |
| 嘎嘎降AI等在线工具 | 付费、非skill生态、不可定制 |

**我们的机会**: humanizer-zh 证明了需求（221安装），但产品质量极差（英文直译），从未更新。我们做真正的中文原生版。

## 三层产品架构

### 第一层：检测（告诉你哪里有AI味）

50+ 条中文AI痕迹规则，按严重程度分级：

#### 词汇层（致命级）
- 体制词汇堆砌：赋能、助力、深耕、沉淀、闭环、打通、抓手、对齐
- 总结陈词套话：综上所述、让我们拭目以待、值得深思、任重道远
- "高级感"假词：深度剖析、全面赋能、精准触达、沉浸式体验
- 成语堆砌：一段话3个以上成语
- 四字词偏好：能两个字说的非用四个字

#### 句式层（明显级）
- 对偶强迫症："既要X又要Y"、"不仅...更是..."
- 排比三连："从A到B，从C到D，从E到F"
- 万能开头："在当今...的大背景下"、"随着...的不断发展"
- 机械过渡："首先...其次...再者...最后..."
- 数字套路："三个维度"、"五大特征"、"七个方面"

#### 情感层（明显级）
- 零立场：永远中立，不敢有观点
- 假共情："相信大家都有这样的感受"
- 过度正能量：每段必须给"积极展望"
- 万能结尾："未来可期"、"让我们共同期待"

#### 结构层（轻微但累积致命）
- 工整病：每段字数差不多，3-4段一组
- 总分总：开头总结+中间展开+结尾呼应
- 小标题过于整齐："一、二、三"齐刷刷
- 段落长度均匀：没有长短错落

#### 标点层
- 分号癖：一句话用3个分号
- 书名号滥用
- 省略号缺失（AI不太用省略号，但真人爱用）

### 第二层：改写（按目标平台风格化）

```
/qinbi 公众号    → 钩子开头+讲故事+制造情绪+金句收尾
/qinbi 知乎      → 有立场+举真实案例+逻辑严密但不死板
/qinbi 小红书    → 碎碎念+具体细节+时间线+真实情绪
/qinbi 汇报      → 结论先行+数据说话+不废话
/qinbi 朋友圈    → 短+有温度+故意留白+不完整
/qinbi 通用      → 去掉最明显AI痕迹，保留内容
```

每个平台的改写规则：

| 平台 | 好文章特征 | AI最容易露馅的点 |
|------|-----------|-----------------|
| 公众号 | 有钩子开头、口语化、会讲故事、有情绪起伏 | 开头太正式、没有悬念、每段都平均 |
| 知乎 | 有立场、有争议、举真实案例、承认不确定性 | "正确的废话"、没有自己的观点、每个观点都"一方面...另一方面..." |
| 小红书 | 碎碎念、有具体细节、有真实情绪、像日记 | 太书面、没有时间线、没有"我"的视角 |
| 工作汇报 | 数据说话、结论先行、不废话、有行动项 | 太多铺垫、"首先其次最后"、没重点没结论 |
| 朋友圈 | 短、有温度、不完整 | 太完整、太逻辑、像广告文案 |

### 第三层：风格指纹（终极壁垒，V2.0）

用户输入自己之前写的3-5篇文章，系统提取个人写作指纹：
- 常用口头禅和语气词
- 段落节奏偏好（长短交错还是均匀）
- 喜欢的连接方式（转折多还是递进多）
- 是否爱用问句、感叹号、省略号
- 用词密度（口语多还是书面多）

改写时自动注入个人风格 → 真正做到"像你自己写的"

## V1.0 MVP 范围

先做第一层+第二层，不做第三层：
1. 中文AI痕迹检测（50+规则）
2. 5种平台风格改写
3. 输出格式：标注问题 + 改写后全文

- 风格指纹系统
- 规则库持续扩充（目标200+）
- 用户反馈收集（哪些改写好/不好）

## 差异化总结

| 维度 | humanizer-zh（竞品） | 亲笔写的（我们） |
|------|-------------------|-----------------|
| 规则来源 | 英文维基百科翻译 | 中文原生，从零构建 |

| 平台区分 | 无 | 5种平台风格化改写 |
| 更新频率 | v1.0.0 从未更新 | 持续迭代 |
| 风格定制 | 无 | V2.0 个人指纹 |
| 数据文件 | 无 | 附带规则库 JSON |
